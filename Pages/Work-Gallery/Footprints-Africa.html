Footprints Africa

Concept

Designed by MUTI

I created this vibrant set of illustrations for Footprints Africa - a non-profit organisation working to advance sustainable development of local African economies.

The colour palette and visual style are inspired by this header image for the page, by MUTI design studio (left). The goal was to create some smaller images to replace stock photos in the card elements, to give the page a more cohesive feel.

I sampled my colour palette form the existing MUTI illustration, aiming to keep this cohesive with MUTI's image. Using the same shadow and highlight colours really helped to inform the way I defined shape in my own graphics.
